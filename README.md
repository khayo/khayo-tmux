# Instalação TPM
Para que tudo funcione corretamente é necessário instalar o gerenciador de plugins TPM
```bash
git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
```

# Ctrl + a + ?
mostra teclas de atalho do sistema

# Ctrl + a + I
instala plugins
